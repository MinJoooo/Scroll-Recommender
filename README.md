# Scroll-Recommender
This program is a program for collecting data on the time the user stays on the screen<br>
while using SNS, analyzing it, and recommending data of interest.<br>
<br>
<br>

관심 데이터를 추천하는 프로그램을 개발하였습니다.<br>
